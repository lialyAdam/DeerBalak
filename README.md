Project Overview

Deer Balak Alena is an emergency management system in Palestine designed to help residents receive alerts about attacks, bombings, protests, closures, and checkpoints. The system also verifies news to prevent the spread of misinformation. It guides users to safe routes, avoids congestion, and enhances overall public safety across various platforms. The interface is user-friendly, interactive, and provides real-time updates.

Features

Real-time emergency alerts and notifications for attacks, protests, bombings, and closures.

Resource tracking and management during emergencies.

Fake news detection and verification to prevent misinformation.

Interactive maps and safe route guidance to avoid traffic jams and dangerous areas.

User management and role-based access control.

Report generation and export, providing detailed insights on performance, resources, and risks.

Responsive and intuitive UI, inspired by modern web applications (as seen in the video).
